# ekaterinburg-plan

Карта ген-плана Екатеринбурга

### Установка

Требуется [Node.js](https://nodejs.org/en/), для установки выполнить

```
npm i
```

### Разработка

Dev-сервер с Hot-reload

```
npm run dev
```

## Сборка и запуск

Проект автоматически разворачивается [в Vercel](https://vercel.com/) при push
